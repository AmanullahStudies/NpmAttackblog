# NpmAttackblog

This repository has been set up as a **standard Next.js TypeScript project** using the pages router. The file structure includes:

- `pages/` folder at the root (contains application routes)
- `_app.tsx`, `_document.tsx` for customization
- `styles/` directory for global CSS
- `public/` directory can be used for static assets

You can start the development server with `npm run dev`.
